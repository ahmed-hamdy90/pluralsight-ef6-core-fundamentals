
## Follow Steps to Include EF 6 Tools and EF Design Packages

***NOTE:*** During Course She works to display Steps using Windows, Visual Studio and PMC. The next Steps for using CLI Commands (Using Linux) with Dotnet CLI

1. Navigate to The Root of the Project, The folder with include Solution `.sln` file Then Define EF tool

```SHELL
dotnet new tool-manifest
```

2. Make sure there is generate configuration JSON file

```SHELL
cat .config/dotnet-tools.json 
```

3. Adding EF Tool - use the latest version at the current date with Dotnet 6

```SHELL
dotnet tool install dotnet-ef --version 6.0.136 --local
```

4. Navigate to Data Project with Include DBContext implemenation Then Include `Microsoft.EntityFrameworkCore.Design` package

```SHELL
cd PublisherData/
dotnet add package Microsoft.EntityFrameworkCore.Design --version 6.0.36
```

5. Finally Make sure to Restore and Build project again

```SHELL
cd ..
dotnet restore && dotnet build
```
