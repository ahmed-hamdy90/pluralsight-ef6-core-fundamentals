
## Follow Steps to Include EF 6 Migration file

***NOTE:*** During Course She works to display Steps using Windows, Visual Studio and PMC. The next Steps for using CLI Commands (Using Linux) with Dotnet CLI

1. Navigate to The Root of the Project, The folder with include Solution `.sln` file.

2. Navigate to Data Project with Include DBContext implemenation.

```SHELL
cd PublisherData/
```

3. Create First Migration file

```SHELL
dotnet ef migrations add intial
```

***NOTE:*** The same Previous Two Steps can be One Command, like next

```SHELL
dotnet ef migrations add intial --project PublisherData/PublisherData.csproj --startup-project PublisherConsole/PublisherConsole.csproj 
```
