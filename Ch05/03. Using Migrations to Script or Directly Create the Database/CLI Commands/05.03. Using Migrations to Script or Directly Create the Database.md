
## Follow Steps to Run EF 6 Migration file

***NOTE:*** During Course She works to display Steps using Windows, Visual Studio and PMC. The next Steps for using CLI Commands (Using Linux) with Dotnet CLI Or Use new Dockerfile for Development

#### Option 1: Create SQL file and Run it later on SQL Server

1. Navigate to Data Project with Include DBContext implemenation.

```SHELL
cd PublisherData/
```

2. Run EF tool to generate migration SQL script

```SHELL
dotnet ef migrations script --output GeneratedSQL.sql
```

#### Option 2: Run EF migrations update command on Local Or use New Development Docker

1. Make Sure remove all Previous Database details 

2. Run EF Update comamnd Locally

```SHELL
dotnet ef migrations update
```
3. Run EF Update comamnd through Docker

```SHELL
docker compose -f docker-compose.development.yaml build
docker compose -f docker-compose.development.yaml up
docker compose -f docker-compose.development.yaml exec ef5coreconsole /bin/bash
```
Inside Container 

```SHELL
cd /src
dotnet ef migrations list --project PublisherData/
```
